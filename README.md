# Bienvenidos a la aplicacion Hulk store

para empezar el proyecto necesitamos aplicar los siguientes comandos:

- cd app
- yarn install
  luego
- cd server
- yarn install

## Inicia el backend

para iniciar el backend es necesario correr la semillas

- cd server
- yarn seed:run
  luego
- yarn run build
- yarn start

## Iniciar el frontend

- cd app
- yarn start
